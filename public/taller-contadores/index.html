El programa · 5 horas

4 módulos. 5 horas. 1 sistema real funcionando al cierre del día.

Cero teoría innecesaria. Llegás con Claude Code pre-instalado (te mandamos guía), trabajás con tus propios datos contables y te vas con una mini-web deployada que podés mostrarle a un cliente el lunes.

1h Módulo 01

Fundamentos + tu entorno listo

Qué es Claude Code, en qué se diferencia de ChatGPT y por qué es la herramienta que te conviene dominar. Verificamos que tu setup funciona.

  • Claude Code vs ChatGPT vs Claude web: cuándo usar cada uno
  • Tu entorno: VS Code, Git y Claude Code funcionando
  • Primer CLAUDE.md personalizado para tu estudio contable
  • Conceptos core: agentes, comandos, subagentes (lo esencial)
1h Módulo 02

Pensar tu trabajo en tareas delegables

El mayor salto no es técnico, es de mentalidad. Aprendés a identificar qué de tu trabajo operativo se puede delegar a la IA y cómo pedírselo bien.

  • Mapeo de tus tareas repetitivas (cada uno con sus datos)
  • Prompts efectivos para el trabajo contable
  • Práctica guiada: Claude analiza tu primer Excel real
  • Detección automática de errores e inconsistencias
2h Módulo 03

Construí InformeMax — tu generador de informes

El corazón del taller. Construimos juntos una mini-aplicación que toma el Excel de un cliente y genera un informe ejecutivo automático con análisis y gráficos.

  • Lectura automática de Excel / CSV contables
  • Cálculos, comparativas y detección de anomalías
  • Generación de informe ejecutivo en HTML con gráficos
  • Conexión con tus propios datos (no ejemplos genéricos)
1h Módulo 04

Deploy + tu roadmap personal

Subís tu sistema a internet con una URL real que podés mostrar. Definís 3 automatizaciones concretas para aplicar en tu estudio la semana siguiente.

  • Publicación en GitHub + Cloudflare Pages (URL pública)
  • Configuración de tu propia API key para seguir usando Claude
  • Roadmap: 3 automatizaciones para aplicar desde el lunes
  • Acceso al grupo de WhatsApp de alumnos + 15 días de soporte

Dudas frecuentes

Respuestas directas. Sin vueltas.

¿Necesito saber programar?
No. Claude Code escribe el código por vos — tu trabajo es decirle qué querés en español. El taller está pensado para contadores sin conocimientos técnicos previos.
¿Qué computadora necesito llevar?
Tu notebook personal, Windows o Mac, con al menos 8GB de RAM. Te mandamos una guía simple 5 días antes para que llegues con todo instalado. Si no podés, también tenemos equipos disponibles.
¿Cuál es la diferencia con ChatGPT?
ChatGPT es un chat dentro del navegador — útil para preguntas sueltas. Claude Code vive en tu computadora, lee tus archivos reales, escribe código, se conecta con bases de datos y publica webs. Es una herramienta profesional, no un chatbot.
¿El costo de Claude Code está incluido?
Durante el taller usás una API key provista por nosotros, sin costo extra. Al final del día te enseñamos a configurar la tuya (cuesta entre 5 y 20 USD al mes según tu uso).
¿Puedo traer datos reales de mis clientes?
Sí, y es altamente recomendado. El taller es mucho más valioso cuando trabajás con tus propios Excel, libros IVA o planillas. Todo queda en tu notebook — nada sale de ahí.
¿Qué pasa si me atraso o tengo dudas después?
Incluye 15 días de soporte por WhatsApp post-taller. Además te sumamos al grupo de alumnos donde compartimos recursos y consultas entre todos.
¿Hay descuento para grupos del mismo estudio?
Sí. Si vienen 2 o 3 personas del mismo estudio, tenemos condiciones especiales. Consultanos por WhatsApp.
